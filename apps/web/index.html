Why Pinata?

Pattern Detection

45+ detection patterns across 10 risk domains. Finds SQL injection, XSS, command injection, and more.

AI Explanations

Get natural language explanations of vulnerabilities. Understand the risk and how to fix it.

Test Generation

Automatically generate security tests for detected gaps. Smart variable filling with AI.

Multiple Formats

Output in terminal, JSON, SARIF, HTML, or JUnit XML. Integrate with any CI/CD pipeline.

Interactive Dashboard

TUI dashboard for exploring results. Navigate gaps, view details, track progress.

Pattern Suggestions

Use AI to suggest new detection patterns from your own vulnerable code samples.
